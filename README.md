# AuraBand
Aura Band je pametna narukvica bez ekrana koja pomoću ESP32 i senzora prati puls i stres. Umjesto ekrana koristi LED lampice za obavijesti, a puni se kombinacijom električne i solarne energije. Isti sistem napajanja koristi i moderno dizajnirano kućište, što osigurava duže trajanje baterije. Uređaj je minimalistički i jednostavan za korištenje.
